## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Rritesh005/ritesh-raghav-portfolio.git
```

Navigate to the project:

```bash
cd ritesh-raghav-portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
